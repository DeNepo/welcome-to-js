# Operators

This operator evaluates to `true` if the _left_ value is **smaller than** _or_
**the same as** the right value.

## _less than or equal to_ operator: `<=`

```js
'use strict';

console.log(1 <= 2); // true
console.log(2 <= 2); // true
console.log(3 <= 2); // false
```
