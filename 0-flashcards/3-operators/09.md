# Operators

This operator evaluates to `true` if the _left_ value is **bigger than** the
right value.

## _greater than_ operator: `>`

```js
'use strict';

console.log(1 > 2); // false
console.log(2 > 2); // false
console.log(3 > 2); // true
```
