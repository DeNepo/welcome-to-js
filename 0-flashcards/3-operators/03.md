## _strict equality_ operator: `===`

```js
'use strict';

// you can use this operator to compare primitives directly
console.log('4' === 4);

// or to compare the values stored in variables
let stringFour = '4';
let numberFour = 4;
console.log(stringFour === numberFour);
```
