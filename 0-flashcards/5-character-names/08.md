# Character Names

This **character** is used for block comments and multiplication.

<details>
<summary>flip the card</summary>
<br>

## `*`

- _singular_: asterisk, or star
- _plural_: asterisks, or stars

```js
'use strict';

/*
  block comment
*/

let four = 2 * 2;
console.log(four);
```
