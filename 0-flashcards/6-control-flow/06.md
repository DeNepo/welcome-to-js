# Control Flow

This type of **control flow** is useful for iterating through each character of
a string.

## A For-Of Loop

```js
'use strict';

let userInput = null;

while (userInput === null) {
  userInput = prompt('enter something');
}

for (let character of userInput) {
  alert(character);
}

alert(userInput + '!');
```
